DESCRIPTION

t.rast2strds takes an input raster and an input strds and puts the raster into a new strds, taking temporal information from the input strds.

NOTES

Temporal databases stored in other mapsets can be used as long as they are in the user's current mapset search path (managed with g.mapsets).
